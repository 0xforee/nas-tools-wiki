# 搜索框

- 可直接在搜索框中输入名称检查资源。
- 左侧图标`T`代表搜索themoviedb的数据，`D`代表搜索豆瓣的数据，`P`代码搜索themoviedb的人物。
- 右侧图标打开高级搜索，高级搜索可输入过滤条件，但不进行识别和严格匹配，直接返回站点搜索结果。
